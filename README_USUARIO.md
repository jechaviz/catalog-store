# Generador de Catálogo Natura

Un generador profesional de catálogos de una sola página que permite crear catálogos visuales hermosos para productos Natura.

## Características

- **Editor de productos en tiempo real**: Ingresa datos de productos en formato de texto
- **Preview visual**: Ve cómo se vería cada página del catálogo mientras editas
- **Edición completa**: Modifica nombre, marca, tamaño, precio, imagen, promociones y más
- **Exportación HTML**: Descarga un archivo HTML completo de una sola página listo para imprimir o compartir
- **Diseño elegante**: Paleta de colores cálidos (rojo terracota, naranja, verde bosque) con tipografía profesional

## Cómo usar

### 1. Agregar Productos

Hay dos formas de agregar productos:

#### Opción A: Formato de texto (recomendado para múltiples productos)

En el panel izquierdo, ingresa productos en este formato, separados por tabulaciones:

```
Nombre | Marca | Submarca | Tamaño | Cantidad | Precio | Revista | Descuentos | Promociones
```

**Ejemplo:**
```
Crema de Manos Granada y Almendras Glaseadas	NATURA	EKOS	50ml	1	$590.00		+$500 compra mayor a $590.00
Body splash Granada y Almendras Glaseadas	NATURA	EKOS	200ml	1	$699.00		KIT COMPLETO
```

Luego haz clic en **"Agregar Productos"**.

#### Opción B: Editor individual

Haz clic en el ícono de lápiz (✏️) junto a cualquier producto para editar:
- Nombre del producto
- Marca y submarca
- Tamaño
- Cantidad
- Precio
- Revista
- Descuentos
- Promociones
- **URL de la imagen del producto**

### 2. Navegar entre productos

- Usa los botones **Anterior** y **Siguiente** en la barra inferior
- O haz clic directamente en un producto en la lista de la izquierda
- El preview se actualiza automáticamente

### 3. Editar productos

- Haz clic en el ícono de **lápiz azul** para editar un producto
- Modifica cualquier campo
- Agrega una URL de imagen para mostrar el producto
- Haz clic en **"Guardar Cambios"**

### 4. Eliminar productos

- Haz clic en el ícono de **papelera roja** junto a cualquier producto
- El producto se eliminará inmediatamente

### 5. Descargar el catálogo

Haz clic en **"Descargar HTML"** para obtener un archivo HTML completo que:
- Contiene todas las páginas del catálogo
- Es de una sola página (cada producto es una página)
- Se puede abrir en cualquier navegador
- Se puede imprimir directamente
- Incluye todas las imágenes y estilos

## Formato de datos

### Campos obligatorios
- **Nombre**: Nombre del producto
- **Marca**: Marca principal (ej: NATURA)
- **Submarca**: Sublínea (ej: EKOS)
- **Tamaño**: Tamaño/volumen (ej: 150ml, 50g)
- **Cantidad**: Cantidad por unidad (ej: 1)
- **Precio**: Precio en formato $XXX.XX

### Campos opcionales
- **Revista**: Nombre de revista o catálogo
- **Descuentos**: Información de descuentos
- **Promociones**: Promociones especiales o ediciones limitadas
- **Imagen**: URL de la imagen del producto

## Diseño del catálogo

El catálogo generado incluye:

- **Fondo cálido**: Gradiente crema y naranja suave
- **Colores principales**: Rojo terracota (#C1440E), naranja (#E8A87C), verde bosque (#2D5016)
- **Tipografía elegante**: 
  - Playfair Display para títulos (serif elegante)
  - Montserrat para categorías y precios (sans-serif moderno)
  - Lato para descripción (sans-serif legible)
- **Elementos decorativos**: Formas orgánicas redondeadas, bordes suaves
- **Layout**: Imagen del producto a la izquierda, información a la derecha

## Ejemplo de uso completo

1. Abre el generador
2. En el área de texto, ingresa:
   ```
   Crema de Manos	NATURA	EKOS	50ml	1	$590.00		+$500 compra mayor
   Body Splash	NATURA	EKOS	200ml	1	$699.00		KIT COMPLETO
   ```
3. Haz clic en "Agregar Productos"
4. Para cada producto, haz clic en el ícono de lápiz
5. Agrega la URL de la imagen del producto
6. Haz clic en "Guardar Cambios"
7. Navega entre productos con los botones Anterior/Siguiente
8. Cuando esté listo, haz clic en "Descargar HTML"
9. Abre el archivo descargado en tu navegador
10. Usa Ctrl+P (o Cmd+P en Mac) para imprimir o guardar como PDF

## Consejos

- **Imágenes**: Usa URLs de imágenes de alta calidad (PNG o WEBP recomendado)
- **Precios**: Usa el formato $XXX.XX para consistencia
- **Promociones**: Usa mayúsculas para destacar (ej: "EDICIÓN LIMITADA!!!")
- **Navegador**: El catálogo se ve mejor en navegadores modernos (Chrome, Firefox, Safari, Edge)
- **Impresión**: Configura márgenes a 0 en las opciones de impresión para mejor resultado

## Soporte técnico

Si tienes problemas:
- Asegúrate de que las URLs de las imágenes sean válidas
- Verifica que los precios estén en formato correcto ($XXX.XX)
- Prueba con diferentes navegadores
- Limpia el caché del navegador si ves cambios antiguos

¡Disfruta creando hermosos catálogos!
